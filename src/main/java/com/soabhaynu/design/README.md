# `com.soabhaynu.design` - Pattern Notes

This folder contains the learning notes (and eventually Java implementations) for each GoF design pattern.

Pattern categories:
- `creational/`
- `structural/`
- `behavioral/`

As you progress, add one file per pattern under the correct category folder (e.g. `FactoryMethod.md`, `Observer.md`, etc.) and keep the structure consistent:
1. Intent and when to use it
2. Participants / key responsibilities
3. Java-specific considerations (thread-safety, immutability, etc.)
4. A small code sketch
